Symfony 3.4 and 4.0 were released the same day — November 30, 2017. That's not a coincidence; it's the strategy.

3.4 is not a feature release. It ships with exactly the same features as 3.3 plus the new deprecation warnings from 4.0. Its purpose is to be the migration tool: upgrade from 3.3 to 3.4, fix every deprecation warning in your logs, then step to 4.0 cleanly.

## :bridge_at_night: Why LTS releases matter in Symfony's model

Symfony releases a new minor version every six months. That pace would be exhausting for production applications to follow, so the project designates every fourth minor as an LTS: three years of bug fixes, four of security fixes. The practical effect is that teams can target 3.4 and not think about upgrades for years.

3.4 is the last LTS of the 3.x line. If you're still on Symfony 2.x or early 3.x, 3.4 is the landing zone.

## :warning: The deprecation layer

Every feature that 4.0 removes is deprecated in 3.4. Run your application on 3.4, enable the deprecation notices, and your logs tell you exactly what to fix. The common ones:

- Services without explicit visibility (public/private) generate warnings — 4.0 makes all services private by default
- `ControllerTrait` is deprecated in favor of `AbstractController`
- The old security authenticator interfaces are marked for removal
- YAML-only service configuration without autowiring annotations triggers warnings

The workflow Symfony intended: upgrade to 3.4, run the test suite with deprecation notices as errors (`SYMFONY_DEPRECATIONS_HELPER=max[self]=0` in PHPUnit), fix everything that fails, then upgrading to 4.0 is mechanical.

## :calendar: The support window

3.4 LTS receives bug fixes until November 2020 and security fixes until November 2021. That's a long runway for production applications that can't follow every release. The cost is staying on the 3.x architecture — no Flex, no micro-framework structure, no zero-config autowiring by default.

The bridge is there. Whether and when you cross it is a business decision, not a technical one.
